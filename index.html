Mobile Security & Privacy Simulator Geo Tool

The Mobile Security & Privacy (MoSP) Simulation Geo Tool helps to modify and optimize spatial data from the OpenStreetMap project for the use with the Mobile Security & Privacy Simulator.

MoSP Geo Tool

  • shows XML-based OSM maps with map tiles in background
  • visualizes the different partitions of a map
  • connects or removes partitions of the map graph based on user parameters (otherwise simulated people could not be routed between road network partitions)
  • implements basic street filters
  • connects selected points of interest (poi) to the road network (basic direct connection as well as intelligent connection of poi inside a building via its entrance)
  • calculates and applies line generalization to the map graph using the Douglas-Peucker algorithm to lower number of nodes based on user parameters

German detailed information can be found in the corresponding bachelor thesis. Simply ask for the pdf document.
